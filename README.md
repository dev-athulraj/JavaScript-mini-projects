# JavaScript Mini Projects

A collection of beginner to intermediate JavaScript projects built with vanilla HTML, CSS, and JavaScript. Each project is self-contained in its own folder.

---

## Projects

| # | Project | Description |
|---|---------|-------------|
| 1 | [Pixel Art Generator](./pixel-art-generator/) | Draw pixel art on a configurable grid using a color picker |
| 2 | [Text Editor](./text-editor/) | A minimal browser-based text editing tool |
| 3 | [Password Generator](./password-generator/) | Generate secure passwords with custom rules |
| 4 | [XO Game](./xo-gaming/) | A premium-styled Tic-Tac-Toe game with score tracking |

---

## Getting Started

No installation or build tools required. Each project runs directly in the browser.

```bash
# Clone the repository
git clone https://github.com/dev-athulraj/JavaScript-mini-projects.git
```
